# 🍽️ Tenpos – Sistema POS en tiempo real

Sistema de punto de venta (POS) en tiempo real diseñado para optimizar la gestión de pedidos en restaurantes, permitiendo la comunicación instantánea entre dispositivos móviles, cocina y caja.

---

## 📌 Descripción

Tenpos es una aplicación web que permite registrar pedidos en tiempo real y distribuirlos automáticamente a las áreas correspondientes (cocina y caja). Su objetivo es mejorar la eficiencia operativa en restaurantes mediante un flujo de pedidos más rápido y organizado.

---

## ⚙️ Funcionalidades

* Registro de pedidos en tiempo real
* Envío automático de órdenes a cocina y caja
* Gestión básica de productos
* Interfaz intuitiva para el personal del restaurante
* Sincronización en tiempo real entre dispositivos

---

## 🧠 Tecnologías utilizadas

* React
* Node.js
* Express
* MongoDB Atlas
* Socket (si lo usas, si no se puede omitir)

---

## 🏗️ Arquitectura

El sistema está dividido en:

* **Frontend:** Interfaz de usuario para toma de pedidos
* **Backend:** API REST para gestión de datos
* **Base de datos:** Almacenamiento en MongoDB Atlas

---

## 🚀 Instalación y ejecución

### 1. Clonar el repositorio

```bash
git clone https://github.com/Joseeee0909/tenpos2.git
```

### 2. Instalar dependencias

```bash
npm install
```

### 3. Ejecutar proyecto

```bash
npm run dev
```
